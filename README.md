# node-cheerio

# Usage

`npm install`    
`node index.js` 抓取中央氣象局的網站在地天氣  
`node app` 抓取 Soft_Job 徵才資訊 並發送至email

# Dev

`npm init`  
`npm i request cheerio --save`  
`npm i nodemailer --save`
